# Jacob Slunga's Portfolio

A modern, minimalist portfolio website showcasing my work as a Computer Science student and Software Engineer. Built with a focus on clean design, smooth animations, and exceptional user experience.

## ✨ Features

- **Responsive Design** - Seamless experience across all devices
- **Dark/Light Mode** - Automatic theme switching with smooth transitions
- **Smooth Animations** - Powered by Framer Motion for buttery interactions
- **Modern UI** - Clean, professional design with subtle gradients
- **Interactive Components** - Engaging hover effects and micro-interactions
- **Fast Performance** - Optimized with Vite for lightning-fast builds

## 🎨 Design Philosophy

The design embraces a **soft, pastel aesthetic** with a cohesive purple-pink color palette throughout. Every element is crafted to feel modern yet approachable, with careful attention to typography, spacing, and visual hierarchy.

### Color Palette

- **Primary Gradient**: Soft purple to pink transitions
- **Backgrounds**: Subtle pastels with low opacity
- **UI Elements**: Glass-morphism effects with backdrop blur
- **Typography**: Figtree font family for excellent readability

## 🛠️ Tech Stack

- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Lucide React & React Icons
- **Routing**: React Router

## 📱 Pages

- **Home** - Hero section with gradient text and action cards
- **About** - Personal story, background, and journey
- **Work** - Professional experience and project showcase
- **Contact** - Get in touch section

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/jacobslunga/portfolio-site.git

# Navigate to the project
cd portfolio-site

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
