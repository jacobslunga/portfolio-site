# Jacob Slunga - Portfolio

A modern, terminal-inspired portfolio website showcasing my projects, experience, and skills as a Computer Science student and developer.

## 🚀 Built With

- **React** + **TypeScript** - Frontend framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling
- **Shadcn/ui** - UI components
- **React Icons** - Technology logos
- **Framer Motion** - Animations

## 🎨 Features

- Terminal-themed design with retro aesthetics
- Responsive layout for all devices
- Scramble text animations
- Dark/light theme toggle
- Project showcase with live demos
- Experience timeline
- Skills overview with tech icons
- Interactive contact section

## 🛠️ Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📱 Sections

- **About** - Introduction and background
- **Projects** - Featured work and applications
- **Experience** - Professional and academic experience
- **Skills** - Technical expertise with visual icons
- **Contact** - Get in touch via terminal interface
